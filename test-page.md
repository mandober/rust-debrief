## Jax

Sets are usually denoted with an uppercase letter, e.g. the set $$\{a,b,c\}$$ may be declared as $$A=\{a,b,c\}$$, and now we can use $$A$$ to stand in for the set $$\{a,b,c\}$$.


$$$
\{n \in \mathbb{N} : n^2 < 20\} = \{ 1, 2, 3, 4 \}
$$$
