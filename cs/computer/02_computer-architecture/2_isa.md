# Instruction Set Architecture

ISA defines:
* states:
  - registers
  - cache
  - main memory
* data types:
  - 6-bit, 16-bit, 32-bit
  - word
  - quadword
* instructions
  - mnemonics
  - translations
  - assembly
* memory consistency
* memory addressing modes
  - immediate
  - relative
  - absolute
* IO model
