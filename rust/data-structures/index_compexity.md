# Data structures: time and space complexity
Time and space complexity of various data structures in big O (asymptotic) notation.


## Worst case

DS        | Space   | Search   | Insert   | Delete
----------|---------|----------|----------|---------
BST       | O(n)    | O(n)     | O(n)     | O(n)


## Average case


DS        | Space   | Search   | Insert   | Delete
----------|---------|----------|----------|---------
BST       | O(n)    | O(log n) | O(log n) | O(log n)
